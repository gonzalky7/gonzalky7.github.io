Getting an API key

  1. The first step is to go to the BreweryDB website and acquire an API key.

  2. After you have created an account you will Agree to Terms of Service you will then be prompted to Register A New App. After that you will have to fill in some information regarding your app only the Name and Description are required.

  3. Then success!! You have acquired your first API key.
