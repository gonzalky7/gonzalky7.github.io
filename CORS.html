How to fix Allow-Control-Allow-Origin

So important before you continue, this explanation is only for the Chrome browser. If you are having issues with Safari or Firefox you will have to try and find the solution yourself.

Follow the video to see if this is the issue

So if you are using Chrome and you followed the video's steps to see if the same error appeared for you, you are having a Cross-origin HTTP request (CORS) problem. A resource makes a cross-origin HTTP request when it requests a resource from a domain or port which is different from the one which the first resource itself serves. What happened with JSFiddle you were making a HTTP request req.open('GET', 'https://api.brewerydb.com/v2/search?q=Goosinator&key=3de3cfdc91affc429fbc0d020361a489&format=json', false); from another domain that it does not recognize. To fix this in Chrome you must add an extension. Click here to add the extension.
